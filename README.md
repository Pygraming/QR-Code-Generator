# QR-Code-Generator
This project allows you to create a qr code with your own text in it.
After the code is created you can save your qr code as png file in a directory put in the location you are at at the moment of running the programm.

Libraries used:
- os ( built-in )
- tkinter ( built-in )
- pyqrcode ( install by typing "pip install pyqrcode" in your terminal/cmd/powershell etc. )
- pypng ( install by typing "pip install pypng" in your terminal/cmd/powershell etc. )
